# ✨ Lyrical AI - Frontend

The modern user interface for Lyrical AI, built with React, Vite, and tailwind. It features a custom streaming reveal for AI-generated lyrics.

## 🛠 Prerequisites
- **Node.js** (v18 or higher)
- **npm**

## 🚀 Setup & Installation

1. **Clone and Enter**:
   ```bash
   cd Lyrical_Frontend-AI_multilingual_lyrics_generator
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

## 🏃 Running the App
1. Ensure the **Backend** is already running (see Backend README).
2. Start the development server:
   ```bash
   npm run dev
   ```

- **App URL**: http://localhost:5173
